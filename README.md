 Breast Cancer Detection using Machine Learning

Overview
This project focuses on predicting whether a tumor is **benign or malignant** using machine learning techniques. The system is trained on a medical dataset and aims to assist in early detection of breast cancer, which is critical for improving survival rates.

Breast cancer is one of the most common cancers worldwide, and early detection plays a crucial role in successful treatment. :contentReference[oaicite:0]{index=0}

---

 Objective
- Classify tumors into **Benign (Non-Cancerous)** or **Malignant (Cancerous)**
- Build a reliable machine learning model for medical prediction
- Perform data preprocessing, feature selection, and model evaluation

---

 Dataset
- Dataset: Breast Cancer Wisconsin Dataset
- Contains features like:
  - Radius
  - Texture
  - Perimeter
  - Area
  - Smoothness
- Target variable:
  - Diagnosis (M = Malignant, B = Benign)

---

 Machine Learning Models

 Support Vector Machine (SVM)
- Used for classification of tumors
- Effective in high-dimensional data

 K-Nearest Neighbors (KNN)
- Simple and effective classification algorithm
- Used for comparison with SVM

Result
- Achieved high accuracy in classifying tumors
- Model performs well on unseen data
- Demonstrates the effectiveness of ML in medical diagnosis
